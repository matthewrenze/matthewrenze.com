Machine Learning: Data Science to AI with R and Python

Data science, machine learning, and artificial intelligence are taking over the world. However, most professionals have not yet learned how to leverage the power of these amazing new technologies. What's stopping you? Now is your chance to learn from one of the leading experts in the IT industry.

In this 3-day workshop you will learn about machine learning from data science to artificial intelligence. You will gain a high-level understanding of the key concepts in these three respective skillsets. In addition, you'll get hands-on experience working with key industry tools like R and Python.

By the end of this workshop, you'll have the skills necessary to solve common business problems with data science, machine learning, and AI.

Workshop Outline

Day 1

  1. Intro to Data Science
  2. Intro to R and Python
  3. Working with Data
  4. Data Analysis
  5. Data Visualization
  6. Hypothesis Testing
Day 2
  1. Intro to Machine Learning
  2. Classification
  3. Regression
  4. Clustering
  5. ML in Practice
Day 3
  1. Intro to Artificial Intelligence
  2. Ensemble learning
  3. Deep learning
  4. Reinforcement learning
  5. Deploying to Production
Day 4 (Optional) - full-day project to combine all key concepts into a start-to-finish AI product.
